# 概要
OCI の HPC インスタンスでHPCGを単一ノードで実行し性能のチェックします。

# 対象
Cluster Network でデプロイされた BM.HPC2.36 インスタンス

# 実行方法
計算ノードのどれかにログインし、以下のスクリプトを実行する。
/etc/opt/oci-hpc/hostfile.rdma ファイルにあるノードに対して一台ずつ HPCG が実行されます。
```
$ ./build.sh
```
